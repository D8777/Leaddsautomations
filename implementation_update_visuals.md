# Goal: Enhance Landing Page with Dynamic Visual Elements

The goal is to add life and depth without crossing into gimmicky territory, transforming the page from a "static dark site" to a "refined, living experience" while maintaining the grounded, professional tone established by the copy.

## User Review Required
Please review the proposed enhancements below and click "Proceed" when you are ready to begin the implementation.

## Proposed Changes

### 1. Background Depth & Atmosphere
- Add animated gradient orbs that slowly drift in the hero background (subtle, low opacity using Framer Motion keyframe animations)
- Implement a subtle noise/grain texture overlay across the page using an SVG filter
- Add floating particle dots (very small, very subtle opacity) that drift upward in the hero section
- Consider a subtle animated grid pattern or dot matrix in one section for visual interest

### 2. Enhanced Card & Button Interactions
- Add hover effects on problem/reframe cards: subtle scale (1.02), border glow with accent color, slight lift with increased border brightness
- Enhance CTA buttons with: scale on hover, animated gradient border glow, subtle background shimmer or pulse effect
- Add icon hover animations: gentle bounce, color shift from neutral to accent
- Implement mouse-following highlight effect on interactive elements

### 3. Typography Animation Upgrades
- Add animated text reveal for the main headline (word-by-word fade-in with stagger)
- Implement subtle gradient text effect on key headlines (white to light gray gradient)
- Add animated accent underlines that draw themselves on section titles
- Consider a typing/cursor effect for specific keywords (used sparingly)

### 4. Scroll-Driven Dynamics
- Add a scroll progress indicator bar at the top of the page
- Implement parallax effect on hero background elements (move at different scroll speeds)
- Add subtle scale/fade to cards as they scroll out of view
- Animate accent borders to "pulse" when in viewport

### 5. Section-Specific Enhancements
- **Hero**: Animated gradient mesh background, floating orbs, particles drifting upward, headline word-by-word reveal
- **Problem Cards**: Hover glow effect, icon micro-animations (rotate/pulse on hover)
- **Reframe Section**: Animated connecting lines between the three concepts, or a central node diagram feel
- **Audit Section**: Step numbers that animate/pulse when in view, timeline connector that draws itself
- **Final CTA**: Subtle spotlight/glow effect that follows mouse position

### 6. Refined Animation Settings
- Keep all animations subtle (no fast or jarring movements)
- Use longer durations (0.8s-1.5s) for ambient effects
- Use shorter durations (0.3s-0.5s) for interaction feedback
- Maintain consistent easing curves (ease-out for entrances, ease-in-out for loops)
